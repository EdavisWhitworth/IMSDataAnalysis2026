# IMSDataAnalysis2026

Desktop analysis and figure-production suite for H5 files generated by IMSSoftware2026.

## Current implementation scope

- Imports IMSSoftware2026 H5 files using `/config` + `/iterations` schema.
- Supports mode-aware visualization for:
  - DTIMS
  - Stepped FTIMS (`FTIMS` in source metadata)
  - Sweep FTIMS (`SWEPT_FTIMS`)
  - Stepped VSIMS (`STEPPED_VSIMS`)
  - Sweep VSIMS (`SWEPT_VSIMS`)
- Displays a navigable heatmap and allows cursor/click selection of individual spectra.
- Detects peaks for the displayed spectrum in two modes:
  - All peaks (thresholded by prominence and SNR)
  - Nearest peak to cursor
- Generates a table of peak metrics:
  - SNR (linear and dB)
  - Reduced mobility (Ko) from user-set P and T (voltage from metadata when available)
  - Resolving power and FWHM
- For Stepped VSIMS:
  - Computes Topt(V) overlay from user-changeable parameters
  - Extracts optimized trace from the heatmap along that curve
- Exports:
  - Current spectrum to SVG
  - Current heatmap to SVG
  - Export uses current plot view/axis ranges and omits interactive peak highlighting.

## UI workflow

1. Click **Load H5 File**.
2. Click on the heatmap to select a spectrum.
3. Optionally click on the spectrum to move cursor location.
4. Adjust pressure, temperature, baseline range, and peak thresholds.
5. Choose **All peaks** or **Nearest to cursor**.
6. For Stepped VSIMS, tune gate multiplier/time add/temperature and inspect optimized overlay.
7. Export the current spectrum or heatmap as SVG.
